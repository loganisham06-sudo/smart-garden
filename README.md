# 🌱 Backyard Digital Twin
An IoT-powered self-preserving garden for the busy professional.

## 🚀 Features
- **Real-time Monitoring:** View soil moisture and temp via a web dashboard.
- **Predictive Care:** Automated watering based on twin-simulated thresholds.
- **Low Maintenance:** Designed for ESP32 deep-sleep battery efficiency.

## 🛠️ Tech Stack
- **Backend:** Flask (Python)
- **Frontend:** HTML5/CSS3/JavaScript
- **IoT Hardware:** ESP32, Capacitive Moisture Sensors
- **Deployment:** GitHub + Render

## 📈 Roadmap
- [ ] Integration with OpenWeather API for rain prediction.
